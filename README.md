# Meru Prison Volleyball Team

A modern responsive website representing the Meru Prison Volleyball Team based in Kenya.

## Features
- React, Vite, TailwindCSS (v3)
- Framer Motion animations
- React Router DOM
- Full responsive design

## How to Run

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
